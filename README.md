# Jack's Portfolio

Personal portfolio website built with React.js and Node.js featuring a retro pixel art design.

## Tech Stack

Frontend: React.js, CSS3  
Backend: Node.js, Express

## Installation

```bash
npm install
cd client && npm install
```

## Running the App

Development mode:
```bash
npm run dev
```

Production mode:
```bash
npm run build
npm start
```

Visit http://localhost:3000 (dev) or http://localhost:3001 (production)

## Project Structure

```
├── client/           # React application
│   └── src/
│       ├── components/   # React components
│       ├── App.js
│       └── App.css
├── server.js         # Express server
└── resume.pdf
```

## Available Scripts

- `npm start` - Run production server
- `npm run dev` - Run development mode
- `npm run build` - Build for production

## Deployment

1. Build the app: `npm run build`
2. Deploy to your hosting platform
3. Set start command: `node server.js`
